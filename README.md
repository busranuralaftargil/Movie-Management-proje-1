# Movie Management System

A web-based movie management application developed using Java and Spring Boot.

## Features

- User Authentication & Authorization
- Movie Management
- Actor Management
- Movie-Actor Relationships
- Role-Based Access Control

## Technologies

- Java 25
- Spring Boot
- Spring Security
- Spring Data JPA
- Thymeleaf
- Maven
- H2 Database
